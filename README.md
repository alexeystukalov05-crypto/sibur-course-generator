# AI-генератор образовательных курсов для Сибура

**Отклик на запрос i.moscow/b2bshowcase/sibur**

---

## A — Проблема клиента

Сибур — крупнейший нефтехимический холдинг России с десятками тысяч сотрудников.

**Текущая боль:**

- Методисты вручную переводят регламенты и инструкции в учебные курсы
- Это занимает недели на каждый документ
- Курсы быстро устаревают при изменении регламентов
- Результат не всегда соответствует стандартам SCORM

> Нужна система, которая делает это автоматически — за минуты, а не недели.

---

## B — Наше решение

AI-система на базе Claude API, которая превращает любой внутренний документ в готовый обучающий курс.

**Как это работает:**

1. Методист загружает документ (регламент, инструкцию, политику)
2. AI анализирует структуру и выделяет ключевые знания
3. Система автоматически строит модули, уроки и тесты
4. Курс экспортируется в формате SCORM для загрузки в LMS

**Стек:**

- Claude API (Anthropic) — анализ и генерация
- Python / FastAPI — бэкенд
- SCORM 1.2 / 2004 — совместимость с любой LMS

---

## C — Демо и доказательства

**Что уже работает:**

Прототип генератора курсов - методист вставляет текст документа, система строит полную структуру курса с модулями, уроками и контрольными вопросами.

**Пример входа:**

Регламент по промышленной безопасности (5 разделов, ~800 слов)

**Пример выхода:**

- Курс: "Промышленная безопасность на объектах Сибура"
- 4 модуля, 12 уроков, 4 теста
- Готов к загрузке в LMS

---

## Почему это работает

- **Скорость:** курс из документа — за 2–3 минуты вместо 2–3 недель
- **Качество:** AI структурирует знания по педагогическим принципам
- **Масштаб:** можно обработать всю библиотеку регламентов Сибура
- **Интеграция:** SCORM-совместимость с SAP SuccessFactors, Moodle, iSpring
